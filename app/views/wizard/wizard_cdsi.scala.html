Community Data Sharing Initiative

The Community Data Sharing Initiative (CDSI) allows users to opt-in for sharing their power quality information to the public. You may include as little or as much information as you feel comfortable with.

Please note that all fields refer to the location of the device you registered in "step 3".

Opting-In

To opt-in to the Community Data Sharing Initiative, make sure "Participate in CDSI" is selected. Then, for each field that you would like to share with the public, make sure the checkbox for that field is selected and that a correct value for that field is entered.

When Finished

This is the last step in the wizard. Review the information in previous steps and select "Finish" to save the changes and exit the wizard. Remember, all of this information can later be edited in the "Administration" section of the website.
